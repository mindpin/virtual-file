virtual_file_system
============

RubyGem 实现虚拟文件夹逻辑
=======

## Installation

Add this line to your application's Gemfile:

    gem 'virtual_file_system'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install virtual_file_system
